# VBE_Operationの仕様
## 定義
|定義名|使用用途|値|
|-----|-----|-----|
|VBE_OPERATION|ライブラリの有無を確認する|1|
|VBE_OPERATION_VERSION|ライブラリのバージョン管理に使用|1.0|

## includeライブラリ
|ライブラリ名|include理由|
|-----|-----|
|stdlib.h|ライブラリ内の計算をする関数(abs)を使用するため|

## 型
|型名|使用用途|
|-----|-----|
|VBE_deta|VBEの描画に必要なデータ群を保持するための構造体|

## 関数
|関数名|動作|戻り値の意味|
|-----|-----|----|
|write_pixel|指定の1ピクセルに書き込む|-|
|fill_screen|画面を単一色で染める|戻り値が0以外の場合範囲外アクセス|
|write_line|指定のxyから指定のxyまでの一直線を引く|戻り値が0以外の場合範囲外アクセス|
|write_line|指定のxyから指定のxyまでの一直線を引く|戻り値が0以外の場合範囲外アクセス|
|write_draw_rect|縦、横を指定し縁だけの四角を描画する|戻り値が0以外の場合範囲外アクセス|
|write_fill_rect|縦、横を指定し塗りつぶされた四角を描画する|戻り値が0以外の場合範囲外アクセス|
|write_draw_rect_crd|write_draw_rectを座標で指定するバージョン|戻り値が0以外の場合範囲外アクセス|
|write_fill_rect_crd|write_fill_rectを座標で指定するバージョン|戻り値が0以外の場合範囲外アクセス|
